## 安装与使用

 > 环境要求: 版本要求Node 14.18+ / 16+ 版本以上
 > 建议使用pnpm否则依赖可能安装不上。

- Get the project code

```bash
git clone https://github.com/opendidi/mind.git
```

- Installation dependencies

```bash
cd mind

pnpm install
```

- run

```bash
pnpm start
```


- build

```bash
pnpm build
```
